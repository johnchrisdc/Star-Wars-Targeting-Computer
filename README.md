# Star Wars Targeting Computer
Watch face for Amazfit Pace

Based on the [GreatFit watchface](https://github.com/GreatApo/GreatFit) by [GreatApo](https://github.com/GreatApo)

## Widgets

**Screen On**
 - Battery percentage
 - Date
 - Time
 - Total Distance
 - Today's Step
 
**Screen Off**
 - Battery percentage
 - Date
 - Time
 - Total Distance
 - Today's Step
